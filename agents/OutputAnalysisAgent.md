# OutputAnalysisAgent

Perform simulation output analysis.

Responsibilities:
- Produce Welch-style moving averages for warm-up discussion.
- Use weekly observations and whole-week batch means.
- Compute means, standard deviations, standard errors, t-based 95 percent confidence intervals, CV, and relative precision.
- Compare standard and antithetic estimators.

Deliverables:
- `welch_plot.png`
- `autocorrelation_plot.png`
- `batch_means_summary.csv`
- `variance_reduction_summary.csv`
- report-ready methodology paragraphs.
